# Expert clipboard

## Shortcut
|        Command        | Description         |
|:---------------------:|---------------------|
|  `Shift + Alt + ESC`  | Close Clipboard     |
|   `Shift + Alt + Z`   | Undo Copy Clipboard |
|   `Shift + Alt + R`   | Redo Clipboard      |
|   `Shift + Alt + H`   | Clipboard History   |
|         `F5`          | Exit Program        |
